Webmonitor II Product Sheet

OVERVIEW

This document briefly describes the features of Webmonitor II, a web based way of monitoring and managing your sybase servers. For further information, please read our On    Line  Documentation,  or you may download the demo version of Webmonitoras a compressed (.gz) tar file - follow the installation instructions in the On Line Documentation to install.

Webmonitor II is a web based utility toolkit for Sybase DBA's and Administrators. It is designed to provide Sybase DBAs and Developers many new tools to manage their servers. Webmonitor IIprovides analysis and new ways to view the server environment and helps perform database administration tasks. The software includes a suprisingly large number of easily understood, easily used utilities that will help you do your job.

Webmonitor II has a robust feature set geared specifically to Sybase. We believe that you will find webmonitor useful and that for the price, it cant be beat. We offer a money  back guarantee and respectfully suggest that you try our software before you purchase it.

The product runs on both Windows NT and UNIX servers. A   normal setup is for an administrator to install webmonitor on their local NT  workstation and then to use it as an enterprise manager for all their UNIX  and NT Sybase servers.  Administration and browsing is performed using your  web browser.   Reports can be generated across your enterprise. For example,  the Morning Review facility allows you to see any problems on your servers  when you come in in the morning. This report includes filtered versions  of your errorlogs, confirmation of your backups, and the output of a server  debugger that looks for common sybase server problems.  Webmonitor IIallows  access to this kind of information from your workstation, through a bookmark,  without the requirement to ever telnet to your systems.

The intranet connects your database environment, so it is a simple leap to envision managing your sybase environment from a web browser. There are several big benefits of tools that use web browser / web servers.Firstly, all the action takes place on your servers (where your log files, backup files, and processing is actually occurring) but the only normal interface to your servers is through command line (shell prompts). Web servers are designed to serve information that resides on your servers to remote clients. Secondly, the web is transparent. You can look at information that exists in multiple servers trivially over the web (the url simply changes to point to another machine). Thirdly, there is no client software required. You install and manage your software in one spot, and anybody in your company can use it. This means that you can provide real time system state information customized to your user community with very little effort. Fourth, the web is open, so you can (with some tweaking of your corporate firewall) permit access to your environment to specific IP addresses from outside your company. This means you can check if your batches ran, and even correct / restart them from your home pc, with virtually no security risk and with no custom software required at home. Finally, the web is easy to program - and the use of the Webify.pm perl library (available for free on our web site) allows easy customization of your Webmonitor to include your nightly batches and any other specific monitoring that you are interested in.

Webmonitor IIprovides interfaces to quite a  few of the freely available utilities that have been developed and distributed by  SQLTechnologiesover the past 6 years.  It provides a nice,  easy to use,  interface to the Extended Stored Procedure Libraryand  to the Sybase Perl UtilitiesToolkit.  These tools were built  to automate virtually every task that the author, as an adminstrator, performed  to manage, tune, and install a variety of database servers at numerous companies  where he worked.  Webmonitor II provides a user interface that allows  administrators to perform operations (like moving data from one server to  another) that should be easy but which are actually non trivial because  no simple software is available to assist the implementation of these tasks.   Some of the facilities in Webmonitor II (like the database design  analysis tool and the database comparison tool) are reasonably complex and  offer tremendous value because they are not available in any other sybase  tools (none that i am aware of - commercial or non commercial). Other features  are more mundane (like a color coded object explorer with hyperlinksbetween  objects and a simple front end to sybase's internal auditing), but offer  no less value. Actually - the hyperlink object reporter is very useful -  it allows you to navigate from a stored procedure to the tables and procedures  it calls without requiring complex navigation - so you can follow the flow of the DDL.

Use of the Webify perl library allows users to  easily customize their version of WebmonitorIIto incorporate  existing command line monitors and utilities into one user interface.  For example,  one client designed and implemented a control panel for their custom data  transfers (it took about a day of custom work - total).

WebmonitorIIis inexpensive when compared to  its feature set and when compared to its competition.  This is because I  fundamentally have a philosophy to share code for free.  In fact, the components  (the Webifylibrary, perl, apache, the extendedstoredprocedurelibrary,  and the sybaseutilitiestoolkit) are all freely available.  On the other hand,  Webmonitor II  has been designed as a commercial   package - and contains several man-years of development effort.   Significant  effort has been placed into making it a helpful tool in your sybase environments.   We hope you will consider using Webmonitor II in your enterprise and that  you will spread the word if you are happy with this product.

PRODUCT FEATURES


 WEB / INTRANET BASED


Webmonitor II provides a web browser based front end to your Sybase and Microsoft SQL Server database systems and (with simple customization) to your batch applications. This means that Webmonitor users can be used by clients, support staff, developers, or end users with no client set up. Monitoring, performance reporting, and system tuning information is delivered to users using their existing desktop software.

EXTENSIVE LOGIN AND FEATURE SECURITY

SYSTEMS / SERVER MONITORING


Webmonitor provides an enterprise level stoplight type view of your servers.

MORNING REVIEW REPORTING


Error logs are scanned, Dbcc output checked, Servers are analyzed for common problems. Output is viewable throught the browser

DATABASE COMPARISON


Webmonitor contains a robust facility to compare the DDL and data in any two databases. This is very useful to compare development systems to your QA and Production environments.

SYSTEM DIAGNOSIS


Webmonitor includes a facility to debug your servers. There are many things that can get internally inconsistent within your servers and this facility will find most of them - ranging from simple setup issues to more complicated issues that might relate to system security and miss configuration.

INTERFACE TO THE SYBASE AUDITING SYSTEM


The sybase auditing system is hard to use. Webmonitor simplifies auditing setup and reporting.

DATABASE COPY


Data and DDL copying between databases is a time consuming task. Webmonitor solves this for you.

INTEGRATED CUSTOMIZED REPORTING / APPLICATION MANAGEMENT


You can set up predefined reports for end users to allow access to application status data.

SIMPLE SERVER OVERVIEW


End users can see basic server information through either an English (show users) or technical (sp__who) interface. This allows business users to identify problems before escalating them to Level 2 tech support.

DATA SERVER REPORTING


Many reports that are not available through any other tools are included in the system. These include simple yet useful cross database reports on things like users (login access to databases database by group). Cross server reports will be available in the next release.

EXTENSIVE DDL SCHEMA GENERATION AND REPORTING


Color coded with hyperlinks between your objects

PERFORMANCE OPTIMIZATION


Several performance tuning tools are available with webmonitorincluding a full database showplan analyzer that color codes showplan output and rates your stored procedure based.

DEPENDENCY ANALYSIS


Find cross database relationships between objects using either a simple search method or using a robust object relationship analyzer.

FILTERED ERROR LOG ANALYSIS

BACKUP AND RECOVERY


Webmonitor provides an integrated enterprise wide view of your backups and dbcc's. It also provides a nice server report that contains all the information you need in case of a crash.

SEPARATE GUI AND APPLICATION ARCHITECTURE


Webmonitoris designed with separate User Interface and Functional components. Command line programs are available for all features. Customer designed programs (nightly data feeds, application specific monitoring, and user defined backup scripts) can be integrated into the User Interface with minimal effort.

OPEN ARCHITECTURE


Webmonitor is provided with source code and with documentation on how to extend the application. It is, in fact, designed to be easily extensible so users can monitor and operate programs in a manner specific to their application and system environment. Customization can be performed by the client or by SQL Technologies.
