# Create Image

Generate a project card image (400x300 WebP) using AI and Pillow.

## Steps

1. Add an `image_description:` field to `METADATA.md` with a plain-English description
2. Ask Claude Code to generate the image using Pillow
3. Save as `docs/images/<name>.webp` (400x300, WebP quality 85)

The Prototyper's own image is generated by `bin/generate_image.py`.
