Announcing: Major releases of our Free Database Backup Scripts and Extended System Stored Procedure Library


We are pleased to announce major releases of our widely used, free, Extended System Stored Procedure Library and Database Server Backup Scripts.  These packages have had major feature upgrades, documentation upgrades, and now support both Sybase ASE and Microsoft SQL Server. Complete information on these packages, as well as information on GEM, our new commercial tool for Database Administrators can be found at http://www.edbarlow.com.

The Extended System Stored Procedure Library consists of numerous updated versions of the system stored procedures. These procedures provide considerable new functionality and have well considered output formatting. They allow users, developers, and administrators to see system data in the format they need. This collection has been widely distributed and is recommended for all Sybase ASE and Microsoft SQL Server installations.

Our open source perl Database Server Backup Solution provide a complete Backup and Maintenance solution for "normal" Sybase and Microsoft SQL Server installations. These scripts work on Unix, Linux, and Windows and are a significant upgrade to the simple shell based tools in use at most sites. This Maintenance Plan driven system provides all normal server maintenance (dbcc, update statistics, table reorgs etc) and uses electronic mail to notify you of success and failure of your scripts. A complete log shipping solution is also included. This code has been well tested and is in use in many production environments.

Microsoft SQL Server support has been added to both packages. The same interface will now work on both Sybase ASE and SQL Server.  Supported platforms now include

  • SQL Server 2000
  • SQL Server 2005
  • Sybase ASE 10
  • Sybase ASE 11.9
  • Sybase ASE 12.5
  • Sybase ASE 15
We have also extended our stored procedure library to include Sybase MDA Table support.  Numerous sp__mda procedures have been released that work similarly to the UNIX iostat and vmstat unix utilities.
