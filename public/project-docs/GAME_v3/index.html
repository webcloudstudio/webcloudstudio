🎮 GAME V3 — Project Assistant

Generic AI Management Environment — Dashboard for managing, monitoring, and operating AI-assisted projects.

Quick Start

cp .env.sample .env
# Edit .env — set SECRET_KEY and PROJECTS_DIR
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
bin/start.sh

DatabaseSQLite (WAL mode, 30s timeout)
UIBootstrap 5 dark theme
BackgroundPython threads (scanner, poller, scheduler, log ingestor)
