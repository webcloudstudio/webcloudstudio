Pricing Details

The following is pricing information for the Generic Enterprise Manager.

GEM is low cost software and is not actively 'sold'. We work with professional database administrators who manage large numbers of Sybase and Sql Servers. To qualify as a customer, you should be a system adminstrator, with administrator access to your database servers. GEM is targeted to customers who have many servers. The monitoring and administration methedologies we use scale well, and are designed to automate repetitive tasks like the collection of event logs and the checking of your backup files. We consider a ten server environment to be the minimum number required for a useful implementation of the software.

GEM uses 'Product Specialists' (consultants) as one method of distribution. 'Product Specialists' are paid 30% of the prices listed below, in return for providing the customer with direct installation and production support. Additional consulting fees can be negotiated with the customer for training and other value-added services. If you are interested in learning the software as a 'Product Specialist', please contact us.

Please contact us for a trial license of GEM if you think you are a qualified customer or are interested in becoming a 'Product Specialist'.
