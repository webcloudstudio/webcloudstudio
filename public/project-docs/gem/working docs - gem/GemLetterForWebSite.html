Open Source Enterprise Administration For Database Administrators

A year and a half ago I embarked on a mission to update my Free Sybase Administration Toolkit. My goal was to produce an inexpensive high-value  commercial quality product that was both Vendor Independant and included everything necessary to automate the routine parts of the DBA's Job. I am pleased to announce that this project is approaching completion, and I would like to Pre-Announce availability of the Generic Enterprise Manager a.k.a. GEM.

GEM is the first whole-enterprise administration tool designed specifically for Sybase and SQL Server administrators. GEM permits proactive management of your Databases and of your Windows/Linux/Unix systems. Features include enterprise monitoring and alarming, health diagnostics, infrastructure reporting, and a "swiss army knife" set of command line utilities and system stored procedures. GEM also includes a full featured plan based Server Backup and Maintenance System.

GEM rolls up 15 years of database free software development into a feature rich set of Database management tools. This gives small and mid sized companies the tools that up until now only the very largest companies possessed. GEM provides extensive coverage for your Sybase, SQL Server, and Oracle databases (delayed until release 2) and for your Windows, Linux, and Unix servers. GEM is designed to work with the vendor provided administration utilities to provide an effective systems management environment.

GEM is open source perl and works identically on both Unix and Windows platforms. An open architecture allows for a constant flow of new features and for easy customization to your needs. For example, it is trivial to incorporate your own monitoring and reporting directly into GEM. Design flexibility allows you to respond “can do” to user requests. GEM is designed by and for the working administrator.

GEM provides a full featured, plugin driven, Perl/TK graphical user interface known as the GEM Enterprise Manager. A server monitor and an alarm viewer plugin are provided with the base installation. We intend to continue to develop plugins that mimic current GEM command line facilities - including a Configuration Auditor, a Log Shipping Manager, and a Database Comparator. These plugins will be integrated to provide a single graphical enterprise manager (similar to the Microsoft SQL Server Enterprise Manager) that will work with all your various Databases and Systems.

GEM is a feature rich, low cost package designed to enhance and complement your existing server management infrastructure. GEM is low risk, using a small footprint, centralized, agentless architecture that allows you to install, evaluate, and run with minimal impact.

Currently, the product is finishing Beta Test. We are looking for partners to provide an initial customer base. Partners get extreme discounts in return for acting as reference sites. Prospective partners should have at least 4 database servers (Sybase or SQL Server) and be willing to provide feedback.
