An Open Source Toolkit For Database Administrators

The Generic Enterprise Manager (GEM) is an open source server management tool box designed specifically for Sybase and SQL Server administrators. GEM is a complete server administration toolkit and is our solution to managing a large number of Database Servers without a lot of stress.

GEM features:

  • Server Monitoring And Alarming
  • Configuration Reporting and Auditing
  • Maintenance Plan based Server Backup
  • A "swiss army knife" set of open source command line utilities
  • Our Extended System Stored Procedure Library.

GEM supports all versions of the Sybase & SQL Server Databases and provides Host support for Win32, Linux, and Unix systems. GEM automates the routine parts of server management. It monitors your systems and notifies you of problems before your users do. Configuration Auditing can reveal serious issues before they become a problem. For example, our backup cross checks catch problems like forgetting to re-enable a backup crontab after an upgrade. GEM gives small and mid sized companies the tools that up until now only the very largest companies possessed.

GEM is based on 15 years of database free software development. GEM is open source perl and works identically on both Unix and Windows platforms. An open architecture allows for a constant flow of new features and for easy customization to your needs. It is trivial to incorporate your business-specific monitoring and reporting into GEM. GEM is designed by and for the working administrator.

GEM is a feature rich, low cost package designed to enhance and complement your existing server management infrastructure. GEM uses a small footprint, centralized, agentless architecture that allows you to install, evaluate, and run with minimal impact. This architecture makes GEM a low risk project for you to set up and try on your servers. GEM will work with your current infrastructure management tools.

Currently, GEM has finished two successful rounds of release testing. GEM is now in a final "Open" Beta-2 implementation - the code is complete, the feature list is frozen, and we are focused on building an initial customer base. Initial customers will get extreme discounts in return for providing feedback.
